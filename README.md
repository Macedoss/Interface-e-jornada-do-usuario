### 🎯 [Atividade 1 - Saúde & Bem-estar App](./Atividade%201)

**Tema:** Aplicativo de Plano de Saúde

Desenvolvimento de wireframe para um aplicativo de saúde com foco em experiência do usuário. O projeto estrutura dois caminhos principais de navegação, priorizando:

- ✅ Navegação simples e eficiente
- ✅ Acesso facilitado às funcionalidades

### 🖥️ [Trabalho 2 - Painel Administrativo de Usuários e Permissões](./Trabalho%202)

**Contexto:** Interface desktop para gestão de usuários e permissões em plataforma corporativa

#### 🎯 Objetivo do Módulo
Permitir que administradores de sistemas:
- 🔍 Localizem rapidamente usuários específicos
- 👁️ Visualizem status e permissões
- 🔐 Alterem níveis de acesso com segurança
- ⚙️ Ativem, suspendam ou removam contas
- 📊 Acessem histórico de alterações

#### 👤 Perfil do Usuário
- Administrador(a) de sistemas experiente
- Pouco tempo disponível
- Alta responsabilidade e baixa tolerância a erros

#### 📌 Tarefas Principais
1. Localizar usuário específico e alterar nível de permissão

#### 📌 Tarefas Secundárias
- Filtrar usuários por status ou perfil
- Visualizar detalhes completos de conta
- Confirmar ações sensíveis com segurança
- Acessar histórico ou logs de alterações

#### 🎨 Desafio UX
Projetar uma interface que:
- 📊 Organize grandes volumes de dados com clareza
- ⚠️ Evite erros críticos
- ⚡ Permita leitura rápida e decisão eficiente
- ✔️ Ofereça feedback claro sobre estados e ações
